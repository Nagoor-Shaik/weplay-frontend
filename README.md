# WePlay — Sports Slot Booking Platform

A full-stack sports venue booking application that allows users to 
discover venues, check availability, and book hourly slots without 
a membership.

## 🚀 Live Demo
- Frontend: https://weplay.vercel.app
- Backend: https://weplay-backend.onrender.com

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Framer Motion (animations)
- Axios (API calls)
- React Router DOM

**Backend:**
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA / Hibernate
- MySQL

## ✨ Features
- User registration and login with JWT authentication
- Browse sports venues by location and sport type
- Real-time slot booking with automatic price calculation
- Animated UI with smooth transitions
- Mobile-first responsive design

## 🏃 Run Locally

### Frontend
```bash
npm install
npm start
```

### Backend
```bash
# Requires MySQL running on port 3306
./mvnw spring-boot:run
```

## 📱 Pages
- Home — Sport categories + quick actions
- Venues — Browse all available venues
- Booking — Select date, time and book
- Login / Register — User authentication
